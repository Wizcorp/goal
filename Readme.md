# Goal-NG

## Requirements

* [Go version 1.11 or higher](https://golang.org/dl/)
* [task](https://github.com/go-task/task)

## Installation

```shell
go get -u github.com/Wizcorp/goal
```

## Create a new project

```shell
goal init github.com/org/repo
```
